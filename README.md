# Naturaleza Argentina – Caminatas y actividades al aire libre

## Descripción
Sitio estático construido con Vite que presenta destinos naturales de Argentina. La home abre con un hero ilustrado y llamadas a las secciones principales.

La página de caminatas ofrece una grilla de senderos destacados con buscador en tiempo real para filtrar por nombre.

La sección de actividades reúne propuestas al aire libre con su propio buscador, ideal para planificar una escapada rápida.

## Stack
- Vite
- JavaScript (ES Modules)
- HTML
- CSS

## Features
- Home con hero ilustrado y enlaces a secciones.
- Página de caminatas con buscador en tiempo real.
- Página de actividades con buscador en tiempo real.
- Navegación entre secciones.

## Cómo correr el proyecto
```bash
npm install
npm run dev
```
